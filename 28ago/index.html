Vivus y scroll reveal

Vivus es una biblioteca de JavaScript diseñada para animar SVG (gráficos vectoriales escalables). Permite crear efectos de "dibujo" para las líneas de un SVG, lo que hace que parezca que se están dibujando en tiempo real. Esta animación puede ser útil para mejorar la interactividad y el atractivo visual de un sitio web.

Aside

El elemento aside en HTML5 se utiliza para representar contenido que es tangencialmente relacionado al contenido principal de la página.
